About Hadoop

Apache Hadoop is an open-source framework for distributed storage and large-scale data processing using the MapReduce programming model.

  • HDFS: Hadoop Distributed File System
  • YARN: Yet Another Resource Negotiator
  • MapReduce: Programming model for large-scale data processing
  • Common Utilities: Shared libraries and utilities
